## 行首偏移

`inset-inline-start` 的主要好处在于，您不需要手动处理不同书写方向的样式。
它会根据文本的具体方向自动决定是左侧还是右侧的插入。这使得程序对国际化的支持更方便，能够适应多种语言和文化。
而 `inset-inline-start` 和 `inset-inline-end` 提供了一种更语义化的方式来定义内边距和外边距。

```css
.selector {
  inset-inline-start: 10px;
}
```

## aspect-ratio

用于设置盒子（例如元素或图片）的纵横比。纵横比是宽度与高度的比值，通常表达为“宽度：高度”。通过指定 aspect-ratio，您可以控制元素的大小和比例，而无需手动设置宽度和高度的具体值。

基础用法

```css
selector {
  aspect-ratio: <ratio>;
}
```

```css
.image-box {
  width: 200px; /* 固定宽度 */
  aspect-ratio: 3 / 2; /* 3:2 纵横比 */
  background-color: tomato;
  object-fit: cover; /* 适用于图片 */
}
```

图片的裁切的另一种方式

> clip-path: inset(20px 0)

```html
<img
  src="https://img.alicdn.com/imgextra/i4/O1CN01auwMRx1IgKHntv5p2_!!6000000000922-2-tps-854-802.png"
  width="420"
  data-aplus-clk="x29_1425fef5"
  data-spm-anchor-id="2415a588.2080e547.0.i88.6854788csbU2bd"
  style="
    width: 420px;
    /* 原图比例是 854/802，我们将分母改小，比如改到 600，图片就变扁了 */
    aspect-ratio: 854 / 713; 
    /* 核心：确保图片填充这个新比例并裁切 */
    object-fit: cover;
    /* 控制裁切的位置：center 是上下对称切，top 是只切底部 */
    object-position: center;
    /* display: block; */
"
/>
```

- object-fit 用于指定元素内容应该如何适应其容器的大小。它通常用于 `<img>` 和 `<video>` 等元素，以控制内容在容器中的缩放和裁剪行为。
- object-position 用于控制替换元素（如 `<img>` 或 `<video>`）在其容器中的位置。它通常与 `object-fit` 一起使用，以确定内容在容器中的对齐方式。
