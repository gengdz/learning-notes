# 系统管理

## lsof

* list open files
* 意思是：列出打开的文件。
* 在 Linux 中，"一切皆文件"的理念意味着 lsof 不仅能显示常规文件，还能显示网络连接、设备文件、管道和套接字等。

### 常用选项

* `-i` : network files / internet addresses。 显示网络连接相关的文件
* `-i :3000` : 显示指定端口（如3000）的网络连接
* `-t` terse mode，简洁模式，只输出 PID（进程号），不显示表头和其他详细信息。
* `-p PID` : 显示指定进程ID打开的文件
* `-u username` :

## kill

* `kill -9 PID` 强制杀死进程
* `kill -9 $(lsof -ti:3000)` 杀死指定端口的进程

## ps

process status

查看进程状态

什么是进程：正在运行中的程序。

当程序被系统加载到内存并开始执行后，就变成了进程。

进程会有：

* PID（进程号）
* 占用内存
* 使用 CPU
* 打开的文件
* 网络连接
* 运行状态

ps aux = 查看系统里几乎所有进程，并用详细、人类可读的格式显示

* a all 所有进程
* u user-oriented format 即：用面向用户的详细格式显示。
* x 包括没有控制终端（TTY）的进程

想看资源占用时使用

ps -ef= 显示所有进程的完整信息

* -e every process 所有进程
* -f full-format 完整格式显示
* -o output format 只输入你指定的列

想看 父子关系，完整命令

```bash
# 示例命令
ps aux          # 显示所有进程的详细信息
ps -ef          # 显示所有进程的完整信息
ps aux | grep node  # 查找node相关进程

ps -eo pid,%cpu,%mem,comm | sort -nrk2 | head
```

## 其他

* clear 清屏 或者使用快捷键 (ctrl+L)
* login 登录（说明：linux 有两种界面一种是图形界面(shift+alt+F1)，一种是字符界面(shift+alt+(F2~F6) )
* logout 退出登录
* shutdown 关机
  * shutdown -h 10 10 分钟之后关机
  * shutdown -r 10 10 分钟之后重启
  * shutdown -k 10 10 分钟之后提醒（只发出警告，不关机《闹钟》）
  * shutdown -c 取消关机 或者 ctrl+c
* halt 立即关机
* reboot 立即重启
* su root 切换到超级管理员
* 查看进程 ps -ef | grep '要搜索的进程名' --> ps -ef|grep node
* 详细查看进程 ps -aux|grep '要搜索的进程名'
* 杀死进程 1. kill PID 2.kill -s 9 PID
* 查找端口 netstat -a 或者 netstat |grep 80
