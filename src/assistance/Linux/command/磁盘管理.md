# 磁盘管理

## ls

显示当前目录下的所有文件

* ls -l （或者简写为写成 ll）: 以列表形式显示
* ls -a 显示所有文件
* ls -al 以列表形式显示所有文件

## mkdir

创建目录

```bash
mkdir [-p] dirName
```

-p 代表 parents ，如果父亲们不存在那么递归创建

```bash
mkdir test
```

## tree

列出文件结构。

```bash
tree -I "node_modules|dist"
tree -L 2 -I node_modules

```

* -I 忽略的目录或者文件模式
* -L level 限制目录显示层级。
* --gitignore gitignore 的文件不显示
