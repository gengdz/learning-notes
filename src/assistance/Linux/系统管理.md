# 系统管理

## lsof

- list open files
- 意思是：列出打开的文件。
- 在 Linux 中，"一切皆文件"的理念意味着 lsof 不仅能显示常规文件，还能显示网络连接、设备文件、管道和套接字等。

### 常用选项

- `-i` : network files / internet addresses。 显示网络连接相关的文件
- `-i :3000` : 显示指定端口（如3000）的网络连接
- `-t` terse mode，简洁模式，只输出 PID（进程号），不显示表头和其他详细信息。
- `-p PID` : 显示指定进程ID打开的文件

## kill

- `kill -9 PID` 强制杀死进程
- `kill -9 $(lsof -ti:3000)` 杀死指定端口的进程
