# Prompt

## 提示词工程是什么？

通过精心的提示词设计，以获得模型更好的输出

## 为什么要提示词工程？

普通交互：像是聊天，随机性强
提示工程：像是编程，可控性强

## 怎么写提示词

Prompt = 指令 + 上下文 + 输入数据 + 输出标准 + 示例

| 要素               | 说明             | 对应范例                                        |
| :----------------- | :--------------- | :---------------------------------------------- |
| 指令 (Instruction) | 想要模型做什么   | "优化产品英文标题"                              |
| 上下文 (Context)   | 补充背景信息     | "这是一款户外折叠椅，在美国亚马逊销售"          |
| 输入数据 (Input)   | 具体要处理的内容 | "current title: Folding Chair Portable"         |
| 输出指示 (Output)  | 指定输出格式     | "输出一行优化的英文标题，不超过150字符"         |
| 示例 (Example)     | 提供参考样例     | "Lightweight Outdoor Folding Chair for Camping" |

## 核心原则

- 写出明确具体的指令
- 给模型足够的思考空间

### 写出明确具体的指令

#### 技巧1: 使用分隔符

常见的如 ``` ,""" 和 <>

#### 技巧2: 要求结构化输出

#### 技巧3: 正向指令原则

告诉模型要做什么，而不是不要做什么

#### 技巧4: 少样本的提示(Few-shot)

提示词：
请为这款“降噪耳机”写一个电商标题。

参考示例：
输入：儿童保温杯
输出：24h锁温不漏水|萌趣儿童保温杯

#### 技巧5: 角色设定

你是一位经验丰富的电商运营专家，擅长撰写吸引人的产品标题和描述。请xxx

#### 技巧6: 重复强化

在长提示的尾部再次强调关键要求，确保模型遵循指令。

### 给模型足够的思考空间

#### 技巧1: 分步完成任务

请按以下步骤为这款“降噪耳机”写一个电商标题：

步骤1：分析产品特点（降噪、音质、舒适度等）
步骤2：识别目标用户群体（通勤族、学生、专业人士等）
步骤3：结合热门搜索关键词
步骤4：撰写标题并控制长度

#### 技巧2: 先解题，再判断

请先计算订单的实际利润，再判断报价单是否合理

#### 技巧3: 意图分类

判断用户的提问意图属于以下哪一类：产品咨询、价格询问、售后服务、使用指导、其他问题。请根据对话内容进行分类并说明理由。

## 进阶推理模式

### CoT Chain of Thought (思维链)

一种让模型展示推理过程的方法，通过逐步分析来提高回答准确性

示例：
请分析这款户外折叠椅的潜在问题，并按以下步骤给出解决方案：
步骤1：识别产品可能存在的问题（从材质、结构、便携性等方面考虑）
步骤2：评估每个问题的严重程度
步骤3：针对高优先级问题提出具体改进建议

### ToT Tree of Thoughts (思维树)

扩展CoT的进阶方法，通过多路径探索和自我验证来解决复杂问题

示例：
请从多个角度分析这款户外折叠椅的市场机会：
路径1：目标用户群体分析（户外爱好者、家庭用户、露营者等）
路径2：竞争对手分析（价格区间、产品特点、市场份额等）
路径3：市场趋势分析（环保材料、智能化功能、个性化定制等）
最终综合各路径分析结果，给出产品定位建议

### Plan-and-Execution 规划求解

核心思想：先制定计划或收集必要信息，再执行任务，避免模型“冲动输出”。

适用场景：

- 新品类市场调研
- 竞品分析报告生成
- 复杂产品参数对比

### ReAct (Reasoning and Acting) 是一种结合推理与行动的提示词框架，特别适用于需要外部工具或知识库支持的复杂任务

核心理念：

- Reason（推理）：分析当前情况，确定下一步行动
- Act（行动）：执行具体操作（如查询数据库、调用API等）
- Observe（观察）：获取行动结果，作为下一步推理的依据

ReAct示例应用：在电商运营中，ReAct框架可以用于智能客服系统：

- Reason: 分析客户问题类型（产品咨询、售后问题、物流查询等）
- Act: 调用相应API获取产品信息、订单状态或退货政策
- Observe: 获取查询结果，生成针对性回复

可以重复上面的过程。
